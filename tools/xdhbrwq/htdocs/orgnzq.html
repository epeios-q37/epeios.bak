Development suspended !

This is the demonstration page of the Epeios organizer web interface.
It runs on a small server behind an ADSL box ; you may experience some unresponsiveness.
If you provide a User ID which doesn't exist, an account is created with given User ID and password.
If the User ID already exists, you must provide the password provided at creation to have access to corresponding account.
This demonstration should work on all modern graphical Web browser, except Microsoft Edge (unless it's the last version, the one coming with Windows 10 version 1607).
Due to this issue, some data will not be displayed correctly on Firefox.
The interface is somewhat ugly, because I am not proficient regarding HTML and CSS ; however, the interface is easily modifiable by someone who knows HTML, CSS and XSL.
It's only a demonstration : passwords are not encrypted, and all accounts will be regularly erased.
Français

Développement suspendu !

Ceci est la page de démonstration de l'interface web de l'application Epeios organizer.
Elle tourne sur un petit serveur placé derrière une box ADSL ; vous serez peut-être confronté à un certain manque de réactivité.
Si vous fournissez un identifiant utilisateur qui n'existe pas, un compte est créé avec cet identifiant et le mot de passe donné.
Si l'identifiant utilisateur existe déjà, vous devez fournir le mot de passe fourni lors de la création pour avoir accès au compte correspondant.
Cette démonstration devrait fonctionner sur tous les navigateurs Web graphique moderne, à l'exception de Microsoft Edge (sauf la dernière version, fournie avec Windows 10, version 1607).
A cause de ceci, certaines données peuvent ne pas s'afficher correctement dans Firefox.
L'interface est trés rustique, parce que mes compétences en HTML et en CSS sont trés limités ; cependant l'interface est facilement modifiable en connaissant HTML, CSS et XSL.
Il s'agit là seulement d'une démonstration : les mots de passe ne sont pas cryptés, et tout les comptes seront régulièrement effacés.
